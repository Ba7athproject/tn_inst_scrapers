# 🚀 Ba7ath Edge Pro v9.0 - "The Jort & Standalone Edition"

Nous sommes fiers d'annoncer la sortie de la version **v9.0**, une étape majeure dans l'évolution de la suite Ba7ath EDGE. Cette version transforme l'outil en une application professionnelle complète, autonome et prête pour une utilisation intensive.

## ✨ Nouveautés Majeures

### ⚖️ Intégration de l'Investigateur JORT
- **Recherche Avancée** : Filtrez les annonces du Journal Officiel par mots-clés, catégories (CONSTITUTIONS, ACTES, ASSOCIATIONS, etc.) et périodes (Année de début à Année de fin).
- **Moteur Intelligent** : Gestion autonome de la pagination et de l'extraction multi-pages via Chrome/Playwright.
- **Support JortSearch** : Intégration sécurisée de vos identifiants JortSearch (configurables dans l'onglet Paramètres).

### 🏛️ Améliorations TUNEPS
- **Filtres de Précision** : Ajout du filtrage par PME et par statut de résultat (Attribué, Infructueux, Annulé).
- **Logique de Sécurité** : Algorithme de navigation imitant l'humain pour éviter les blocages WAF.
- **Export Excel** : Génération de rapports structurés avec des noms de fichiers horodatés.

### 🎨 Design System "Slate & Azure"
- **UI Card-Based** : Une interface moderne organisée en cartes blanches épurées pour une meilleure lisibilité.
- **Signalisation Dynamique** : Un voyant de statut en temps réel (Vert/Orange) dans le header pour suivre l'activité du moteur d'extraction.
- **Aesthétiques Premium** : Palette de couleurs Slate & Azure Vibrant, typographie Segoe UI moderne et boutons affinés.

### 🍱 Standalone & Portabilité
- **Plus besoin de Python** : L'application est désormais compilée en un seul fichier `.exe` indépendant.
- **Assets Bundlés** : Le logo, l'icône et la base de données des acheteurs sont tous intégrés dans le package final.
- **Build un clic** : Utilitaire `build_standalone.py` inclus pour les futures mises à jour.
